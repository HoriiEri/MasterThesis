ここでは本論文で紹介したソフトウェア，
Point Cloud Generator（\textgt{図 \ref{fig:PCG_areaexplain}}）の利用方法について記述する．
このソフトウェアはLPSDで描画する物体の点の明度・密度を事前評価することを可能とし，
LPSDに入力可能な点列データを生成する．
また次に示す5つの機能をもち，Mac OS X上で動作する．

\begin{itemize}
\setlength{\itemsep}{-3pt}
\item 3次元ポリゴンモデルのスケール変更機能
\item 3次元ポリゴンモデルからの表面特徴量算出機能
\item シミュレーション機能
\item 最終的な描画ファイルの出力機能
\item ゲージを用いた描画の評価機能
\end{itemize}

\noindent
各機能について\ref{sec:apd_scale}〜\ref{sec:apd_evaluation}で詳しく説明する．
